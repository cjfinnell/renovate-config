# Renovate Configuration

To use this base config, specify the following in your `.github/renovate.json`:

```
{
  "extends": ["github>cjfinnell/renovate-config"]
}
```

https://docs.renovatebot.com/config-presets/

## Secrets

Secrets are managed in the Renovate webapp, using Github to authenticate:
- https://docs.renovatebot.com/mend-hosted/migrating-secrets/
- https://developer.mend.io/github/cjfinnell/-/settings
